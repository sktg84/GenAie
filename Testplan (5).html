Testcase ID: 2.1

Testcase Title

Verify EE module is present in PSN Release 7.0

Test Topology

  • One PSN running Release 7.0

Test Procedure

  1. Connect to the PSN using a terminal emulator.
  2. Enter the "show version" command.
  3. Verify that the output includes a line that says "EE module: present".

Pass Criteria

  • The output of the "show version" command includes a line that says "EE module: present".

Testcase ID: 2.2

Testcase Title

Verify old EE is not present in PSN Release 7.0

Test Topology

  • One PSN running Release 7.0

Test Procedure

  1. Connect to the PSN using a terminal emulator.
  2. Enter the "show version" command.
  3. Verify that the output does not include a line that says "Old EE module: present".

Pass Criteria

  • The output of the "show version" command does not include a line that says "Old EE module: present".

Testcase ID: 2.3

Testcase Title

Verify NOC can control which EE is running on a PSN

Test Topology

  • One PSN running Release 7.0
  • One NOC workstation

Test Procedure

  1. On the NOC workstation, connect to the PSN using a management tool.
  2. Navigate to the EE configuration page.
  3. Select the "New EE" or "Old EE" option.
  4. Click the "Apply" button.
  5. Verify that the EE on the PSN changes to the selected option.

Pass Criteria

  • The NOC can successfully control which EE is running on the PSN.

Testcase ID: 2.4

Testcase Title

Verify EE can establish a connection with a host

Test Topology

  • One PSN running Release 7.0
  • One host

Test Procedure

  1. On the host, start a TCP server on port 80.
  2. On the PSN, enter the "connect" command to the host's IP address and port 80.
  3. Verify that the connection is established successfully.

Pass Criteria

  • The EE can successfully establish a connection with the host.

Testcase ID: 3.1

Testcase Title

Validate PSN interoperability between hosts running old EE and hosts running new EE

Test Topology

  • Two PSNs, one running the old EE and one running the new EE
  • Hosts connected to both PSNs

Test Procedure

  1. Send packets from hosts on the old EE PSN to hosts on the new EE PSN.
  2. Send packets from hosts on the new EE PSN to hosts on the old EE PSN.
  3. Verify that packets are successfully received and processed by hosts on both PSNs.

Pass Criteria

  • Packets are successfully received and processed by hosts on both PSNs.

Testcase Title

Test Topology

  • Two hosts connected to an IMP

Test Procedure

  1. Host A sends a connection request message to Host B
  2. Host B receives the connection request message and sends a connection accept message to Host A
  3. Host A and Host B exchange data messages

  • Host A and Host B can exchange data messages

Testcase ID: 4.3

Testcase Title

Verify AHIP can be used to support multiple concurrent connections between two hosts

Test Topology

  • Two hosts connected to an IMP

Test Procedure

  1. Host A sends a connection request message to Host B
  2. Host B receives the connection request message and sends a connection accept message to Host A
  3. Host A sends a second connection request message to Host B
  4. Host B receives the second connection request message and sends a second connection accept message to Host A

Pass Criteria

  • Host A and Host B can exchange data messages on both connections
